% ReadMe File: How to run Frame Annotation Tool MATLAB GUI

To run the code, follow the steps below,

- Download the repository from Git website. The Zip fie can be downloaded from,

  https://github.com/mmshx/FATGUI/archive/master.zip

  or use git clone command (you need to enter your Git account credentials),

  git clone https://github.com/mmshx/FATGUI.git

- Unzip the downloaded folder.

- Add the folder directory to MATLAB path or change MATLAB path to this folder.

- Type MCC in MATLAB command windows to run the application.

Note the output file is saved in the main folder and the backup files are saved
in Backup_OutPutStruct sub-folder.

If the GUI does not show all elements, resize it.

Enjoy !
